# Dune Hostnames 
## Author: Nate Dolny 

## Description: 

Inspired by the iconic book series Dune, authored by Frank Herbert, 
Dune Hostnames offers a comprehensive array of naming schemes for 
network infrastructure, workstations, servers, and other devices.
